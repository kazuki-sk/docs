# 40_Middleware

## Definition
アプリケーションやサービスを実行・維持するために必要な汎用的なミドルウェアおよびプラットフォーム。

## Scope
* **Container Platform**: Docker Engine, Kubernetes (k3s/k8s), Docker Swarm
* **Web Server / Proxy**: Nginx, Traefik, Apache
* **Database**: PostgreSQL, MySQL, Redis (Server setup)
* **Certificates**: Let's Encrypt (Certbot)
