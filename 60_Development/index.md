# 60_Development

## Definition
ローカルPC（Mac/Windows/Linux Desktop）上の開発環境、ツールチェーン、および個人の生産性向上のための設定。

## Scope
* **Editors**: VSCode (Settings, Extensions), Neovim
* **Version Control**: Git Config, GitHub CLI
* **Languages**: Python (pyenv), Node.js (nvm), Go, Rust environment
* **Dotfiles**: `.zshrc`, `.bashrc` (Personal configs)
