# 20_Virtualization

## Definition
ハードウェア抽象化層、仮想化基盤、およびクラウドプロバイダーのリソース管理（IaC）を行う。

## Scope
* **Hypervisor**: Proxmox VE, ESXi, Hyper-V
* **Container Host OS**: Talos Linux, Flatcar (if applicable)
* **Cloud Infrastructure**: Azure VM Instances, AWS EC2 (Instance level)
* **IaC**: Terraform, Ansible (for Infra provisioning)
