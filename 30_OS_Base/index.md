# 30_OS_Base

## Definition
OS（Linux/Windows）の初期セットアップ、セキュリティ堅牢化、ユーザー管理に関する標準手順。
特定のアプリに依存しない、システム共通の設定をここに記述する。

## Scope
* **Initial Setup**: Netplan (IP設定), Timezone, Locale
* **Security**: SSH Config (Hardening), Firewall (UFW/iptables), Fail2ban
* **User Management**: Sudoers, LDAP/AD Integration
* **Maintenance**: Kernel Update, Auto Upgrades
