# Global Scraps

## Definition
特定のカテゴリ（Network, Virtualization等）に分類しきれない、あるいは分類前の「作業メモ」「調査ログ」「アイデアの殴り書き」を一時的に保管する場所。

## Scope
* **Unclassified**: どのフォルダに入れるべきか迷っている作業ログ
* **Cross-cutting**: 複数のレイヤーにまたがる調査（例：OSとNetworkとAppが全部おかしい時の調査）
* **Drafts**: 新しいトピックフォルダを作る前の下書き、構成案のメモ

## Rules
1. **Filename**: `yyyy-MM-dd-[topic].md` の形式を推奨。
    * 時系列でソートされ、過去の思考プロセスを追いやすくするため。
2. **Lifecycle**:
    * まずはここで気軽に書き始める。
    * 内容がまとまったり、解決策が見つかったら、適切なトピックフォルダ（例: `50_Services/Minecraft/index.md`）へ清書して移動（昇格）させる。
    * 完了後のメモは、ログとしてここに残してもよいし、削除してもよい。
